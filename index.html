SQL enabled Subgraph Services API

SQL enabled Subgraph Services API

graph-sql

Discovery and execution services API for sql enabled graph protocol subgraphs

Highlights

  • Discovery API
  • SQL execution API
  • Package include CommonJS, ES Modules, UMD version and TypeScript declaration files.

Install

npm install @semiotic-labs/graph-sql

Usage

import {createConnection} from '@semiotic-labs/graph-sql';

const connection = await createConnection(
  {
    displayName: 'Graph EBO Arbitrum'
  },
  undefined,
  true
);

 const queryresult = await connection.execute('SELECT 1;');

Gateway API Key

SQL query execution requires API key for sql enabled gateway. Get your API keys from https://sql.semiotic.ai. You can either pass it to createConnect as a parameter or set as GATEWAY_API_KEY environment variable.
